# 📄 changelog.md

## [v1.4.0] - 2026-04-XX

### ✨ 주요 기능 추가

#### 1. 홈 중심 허브형 네비게이션 구조
- 홈 버튼 기준 좌/우 동적 탭 구조 도입
- 일정 / 자산 기능을 각각 독립 허브 형태로 구성
- 탭 진입 시 좌/우 기능 자동 변경
- 홈 버튼 항상 중앙 정렬 유지

---

#### 2. 반복 일정 / 할일 시스템 개선
- 반복 항목을 실제 개별 인스턴스로 처리
- 특정 날짜 기준 개별 완료/상태 변경 가능
- 반복 수정/삭제 시 범위 선택 기능 추가
  - 이번만
  - 이후
  - 이전
  - 전체
- 요일 반복 / 간격 반복 / 종료일 처리 개선
- 반복 종료일 “없음” UI 통합 처리

---

#### 3. 날짜 선택 팝업 기능 확장
- 달력에서 날짜 클릭 시 상세 팝업 표시
- 해당 날짜 항목:
  - 종일 일정 영역
  - 타임라인 영역 (시간 기준)
- 타임라인 현재 시간 표시 기능 추가
- 선택 날짜에서 바로 항목 추가 가능

---

#### 4. 일정 장소 기능 고도화
- 기본 장소 + 날짜별 장소 시스템 통합
- 날짜별 장소 자동 유지 로직 추가
  - 예: 14일 설정 → 이후 날짜 자동 적용
- 일정 기간 중 특정 날짜 기준 장소 변경 가능
- 장소 UI:
  - chip 형태 표시
  - Google Maps 연동 (온라인 시)

---

#### 5. 가계부 시스템 확장
- 지출 → 수익/지출 통합 구조로 변경
- 월 예산 / 사용량 / 잔액 계산 기능 추가
- 카테고리 / 서브카테고리 구조 적용
- 결제수단 / 태그 / 메모 필드 추가
- 자산 관리 기능 추가
  - 주식 / 적금 / 예금
- 자산 자동 반영 (수익 기반)

---

#### 6. OCR 영수증 분석 기능
- 이미지 기반 거래내역 자동 분석
- OCR(Tesseract) 기반 텍스트 추출
- 지출 항목 자동 파싱
- 수익(+금액) → 자산 자동 반영
- 분석 후 사용자 검토 큐 시스템 도입

---

#### 7. 시간 선택 UI 개선
- 커스텀 TimePicker 컴포넌트 구현
- 5분 단위 선택 + 직접 입력 지원
- 검색 기능 포함
- 일정 / 할일 / 팝업 모두 공통 적용

---

#### 8. 요약(대쉬보드) 기능 강화
- 전체 / 완료 / 실패 / 진행 통계
- 달성률 퍼센트 표시
- 오늘 포함 일정 / 긴급 할일 표시
- 연도 / 월 필터 적용
- 페이지네이션 추가

---

#### 9. 인증 및 데이터 동기화
- Firebase Auth 로그인/회원가입 구현
- Firestore 기반 데이터 저장
- Planner / Finance 데이터 분리 저장
- 오프라인 시 localStorage fallback 지원

---

### 🛠 구조 개선 (대규모 리팩토링)

#### 1. 모듈화 구조 도입
- calendar.js → 달력 렌더링
- dashboard.js → 요약 / 통계
- plannerItems.js → 항목 CRUD 로직
- plannerUI.js → UI 제어
- finance.js → 가계부 로직
- financeOcr.js → OCR 처리
- repeat.js → 반복 로직
- storage.js → 로컬/클라우드 저장
- timePicker.js → 시간 UI
- utils.js → 공통 유틸

---

#### 2. 전역 상태 관리 구조 정리
- `window.AppState` 도입
- `window.AppRefs` DOM 참조 관리
- `window.FirebaseRefs` 분리
- 상태 접근 함수 기반 구조로 변경

---

#### 3. 렌더링 구조 분리
- 카드 렌더링 분리 (renderItems.js)
- 캘린더 / 대쉬보드 / 리스트 각각 독립 렌더링
- UI와 데이터 로직 완전 분리
