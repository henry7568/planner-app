# Changelog

모든 주요 변경사항은 이 파일에 기록됩니다.

---

## v0.4.0 - 2026-04-15

### 추가됨 (Added)

- 할일(todo) 기능 추가
  - 기한 날짜 입력 가능
  - 기한 시간 입력 가능
  - 남은 시간 자동 계산 표시

- 일정(schedule) 기능 추가
  - 시작 날짜 입력
  - 종료 날짜 입력
  - 시작 시간 / 종료 시간 입력
  - 기간 일정 관리 가능

- 상태 체크 시스템 추가
  - 빈칸 = 미선택
  - "-" = 실패
  - "✓" = 성공
  - 버튼 클릭 시 상태 순환

- 월별 필터 기능 추가
  - 특정 월 기준으로 일정 및 할일 필터링

- 요약 통계 UI 추가
  - 전체 개수
  - 미선택 개수
  - 실패 개수
  - 성공 개수

---

### 달력 기능 (Calendar)

- 일정 달력 뷰 추가
- 월 이동 기능 추가
- 일정이 있는 날짜에 표시
- 일정 개수 많을 경우 "+ n개 더" 표시

---

### 일정 상세 보기 (Overlay Popup)

- 달력 날짜 클릭 시 상세 창 표시
- 달력 위에 오버레이 팝업 UI 구현
- 팝업에서 일정 정보 확인 가능
- 팝업에서 일정 수정 가능
- 팝업에서 일정 삭제 가능
- 팝업에서 상태 변경 가능
- 팝업 배경 클릭 시 닫기 기능
- 닫기 버튼 추가

---

### 수정 기능 (Edit)

- 기존 항목 수정 기능 추가
- 수정 시 기존 데이터 자동 입력
- 수정 취소 버튼 추가

### 데이터 처리

- localStorage 저장 구조 개선
- 이전 버전 데이터 자동 로드 지원
- 일정 기간 범위 계산 로직 추가

### UI 개선

- 카드 레이아웃 정리
- 일정 상세 카드 확대 UI
- 상태 버튼 스타일 개선
- 모바일 대응 레이아웃 개선
