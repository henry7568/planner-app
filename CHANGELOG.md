### 2. 반복 일정 범위 수정 / 삭제 기능 추가

- 지원 범위:
  - 해당 일정만 적용
  - 해당 일정 이후 적용
  - 해당 일정 이전 적용
  - 전체 일정 적용
- 반복 일정 삭제도 동일한 범위 선택 popup을 사용하도록 정리
- 삭제 버튼 중복 이벤트로 인해 전체 삭제가 먼저 실행되던 문제 수정
- occurrence 카드 클릭 시 occurrence 날짜를 유지한 채 범위 수정 / 삭제 흐름으로 진입하도록 보정

### 3. 반복 일정 구조를 슬롯형 방식으로 조정
- `weekly_days` 반복 일정 저장 시 `repeatSlotDates`, `repeatSlotStatuses` 기반으로 관리하도록 변경
- 요일별 반복 일정은 미래 occurrence를 전부 펼치는 방식 대신 현재 활성 슬롯만 유지하도록 수정
- 특정 요일 일정 완료 시 해당 요일 슬롯만 다음 주 동일 요일로 이동하도록 처리
- 달력 / 오늘 목록 / 대시보드가 슬롯형 반복 일정 구조를 읽도록 렌더 흐름 정리
- 반복 일정 삭제 후 남은 슬롯이 달력에서 유지되도록 삭제 범위 로직 보정

### 4. 선택 날짜 popup 추가 흐름 개선
- 선택 날짜 popup에서 할일 / 일정 추가 시 메인 추가와 동일한 반복 저장 구조를 사용하도록 수정
- 선택 날짜 popup에서 일정 추가 시 `dailyLocations`가 함께 저장되도록 수정
- 선택 날짜 popup에서 추가 후 홈으로 이동하던 버그 수정
- popup에서 추가하는 반복 일정이 이전 반복 방식으로 저장되던 문제 수정
- popup의 할일 / 일정 반복 옵션과 요일별 반복 UI를 메인 폼과 같은 순서와 문구로 맞추는 방향으로 정리

### 5. 반복 종료일 UI 공통 정리
- 메인 추가 / 선택 날짜 popup / 수정창 / 가계부에 공통 적용되는 반복 종료일 UI를 통일
